# Anime Avatar Generator
- A simple web application that transforms selfies into anime-style avatars using AnimeGAN V2.
# The Link to the Demo Web Application
- https://huggingface.co/spaces/andyting0619/Anime-Avatar-Generator
# Requirement
- Python 3.10
# How to Use
1. Download and extract the files.
2. Run the command prompt as an administrator.
3. Install the required packages using the following command:
   
   ```Bash
   pip install -r requirements.txt

4. Run the Python script `anime_avatar_generator.py` to start the web application.
5. Open your web browser and go to `http://127.0.0.1:7860` to access the web application.
